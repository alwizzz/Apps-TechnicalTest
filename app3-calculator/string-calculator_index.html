String Calculator

Rules

  • Calculator can only calculate one operation with two numbers at one time
  • You must enter the operation consisting first number, operator, and second number (in order)
  • Every parts are separated by single space ' '
  • Any extra input will be ignored
  • Entering non number will stop the number parsing e.g. 112a32 -> 112
  • Number can not be more than 1000000 (1 million)

Supported operation: addition (+), substraction (-), multiplication(*), and division(/)
